# The Malawian Fish Room — Website 2

A premium, Marine Collectors–inspired storefront for **The Malawian Fish Room**, built with Next.js and styled for a photography-first, boutique aquarium retailer feel.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for static hosting

```bash
npm run build
```

Static files are exported to `out/` (GitHub Pages–ready). Set `BASE_PATH` if deploying to a project subdirectory.

## Project structure

- `app/` — Pages (home, catalog, about, contact)
- `components/` — Header, footer, hero, product cards, catalog filters
- `data/fish-catalog.json` — Fish inventory from the original site
- `public/photos/` — Logo, Lake Malawi, and fish images
