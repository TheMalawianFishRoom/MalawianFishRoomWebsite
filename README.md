This is a business website built for **The Malawian Fish Room** who is Ontario based African cichlid breeder and fish retailer specializing in premium freshwater fish, educational customer support, and more.

This project was designed to create a premium digital presence focused on:

* professional branding,
* responsive user experience,
* scalable catalog management,
* promotional marketing,
* and long-term business growth.

---

# Project Overview

The website was built as a custom Next.js application using modern frontend technologies and a reusable component architecture.

The project focuses heavily on:

* responsive design,
* clean UI/UX,
* premium visual presentation,
* modular scalability,
* and long-term maintainability.

The site includes:

* a homepage with promotional marketing sections,
* a fish catalog system,
* informational pages,
* contact systems,
* FAQ systems,
* SEO support,
* and reusable content-driven components.

---

# Running The Project Locally

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

The project will run locally at:

```text
http://localhost:3000
```

---

# Production Build

## Create Production Build

```bash
npm run build
```

## Start Production Server

```bash
npm run start
```
